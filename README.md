## Run Locally

Clone the project

```bash
  git clone https://github.com/yoann90/projet-collectif-plateforme-de-vente-de-meubles-NOYALES_Meubles.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
